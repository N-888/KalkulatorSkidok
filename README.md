# Калькулятор скидок для магазина

## Описание проекта

**Калькулятор скидок** — это удобный инструмент для быстрого расчёта скидок в магазине. Программа позволяет ввести цену товара и процент скидки, автоматически вычисляет итоговую стоимость и сохраняет историю всех вычислений.

### Преимущества

- **Простота использования** — понятный интерфейс на русском языке
- **Безопасность** — обработка ошибок ввода, защита от некорректных данных
- **Сохранение истории** — все вычисления записываются в файл
- **Логирование** — все действия фиксируются в файле журналов
- **Минимум зависимостей** — работает только на стандартной библиотеке Python

### Этап 1: Идея

На этом этапе я придумал простую бытовую задачу — калькулятор скидок для магазина. Часто в магазине нужно быстро рассчитать, сколько я сэкономлю при скидке. Я понял, что нужна программа, которая принимает цену товара и процент скидки, а затем показывает итоговую стоимость. Это простая, но полезная задача для повседневной жизни.

### Этап 2: Промпт

На этом этапе я составил подробное задание для AI на понятном русском языке. Я описал все требования: ввод данных, расчёт скидки, сохранение истории, логирование и обработку ошибок. Промпт должен быть максимально понятным и конкретным, чтобы AI понял, что именно нужно сделать. Хороший промпт — это залог качественного результата.

### Этап 3: Код

На этом этапе AI проанализировал мой промпт и сгенерировал программу на Python. Программа была написана с использованием принципов ООП: созданы классы DiscountCalculator и FileManager, добавлены комментарии на русском языке. Код получился чистым, читаемым и готовым к расширению. AI также добавил обработку ошибок и логирование.

### Этап 4: Запуск

На этом этапе я запустил программу и проверил, как она работает. Я ввёл тестовые данные: цену 1000 рублей и скидку 20%. Программа корректно рассчитала итоговую цену — 800 рублей. Я также проверил обработку ошибок: при вводе отрицательной цены программа вывела сообщение об ошибке. Все функции работают стабильно.

### Этап 5: Пользовательский результат

На этом этапе пользователь получает готовый инструмент для расчёта скидок. Программа понятна и удобна в использовании: нужно ввести цену и скидку, а результат отобразится мгновенно. Все вычисления автоматически сохраняются в файл history.txt, а действия логируются в файл logs.txt. Пользователь может использовать программу снова и снова.

## Техническая документация

### Класс DiscountCalculator

Основной класс программы, реализующий расчёт скидок.

**Методы:**

- `calculate_discount(price, discount_percent)` — вычисляет итоговую цену
- `save_history(filename)` — сохраняет историю вычислений в файл
- `get_history()` — возвращает список всех вычислений

### Класс FileManager

Модуль для работы с файловой системой.

**Методы:**

- `create_file(filename, content)` — создаёт файл
- `read_file(filename)` — читает содержимое файла
- `delete_file(filename)` — удаляет файл
- `file_exists(filename)` — проверяет существование файла
- `create_directory(dir_name)` — создаёт папку
- `list_files()` — получает список файлов

## Архитектура проекта

Проект построен с использованием принципов объектно-ориентированного программирования (ООП):

1. **Модульность** — код разделён на отдельные модули с чёткими обязанностями
2. **Инкапсуляция** — данные и методы объединены в классы
3. **Расширяемость** — код готов к добавлению новых функций

## Бизнес-правила

1. **Нельзя редактировать/удалять** при пустом списке вычислений
2. **Цена товара** не может быть пустой или отрицательной
3. **Процент скидки** должен быть от 0 до 100
4. **История** сохраняется автоматически при выходе из программы

## Часто задаваемые вопросы

**В: Как выйти из программы?**  
О: Введите слово «выход» вместо цены товара.

**В: Где сохраняется история вычислений?**  
О: В файле `history.txt` в папке с программой.

**В: Можно ли использовать программу повторно?**  
О: Да, программа работает в цикле и позволяет делать несколько вычислений подряд.
