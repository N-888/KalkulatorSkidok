## О проекте

**Калькулятор скидок** — это простая и удобная программа для быстрого расчёта скидок в магазине. Введите цену товара и процент скидки — программа мгновенно покажет итоговую стоимость и сумму вашей экономии.

### Почему этот проект?

- Мгновенный расчёт скидок за один клик
- Сохранение истории всех вычислений
- Защита от ошибок ввода данных
- Работает на любом компьютере
- Не требует установки дополнительных программ

## Как использовать

### Запуск через командную строку

1. Откройте папку проекта в проводнике
2. В строке адреса напишите `cmd` и нажмите Enter
3. Введите команду `python main.py`

### Запуск через Python IDLE

1. Откройте Python из меню "Пуск"
2. Нажмите File → Open
3. Выберите файл `main.py`
4. Нажмите Run → Run Module

### Запуск двойным кликом

**Для консольной версии:**
1. Зайдите в папку проекта
2. Дважды кликните на файл `Запуск.bat`
3. Откроется окно консоли с программой

**Для графической версии:**
1. Зайдите в папку проекта
2. Дважды кликните на файл `Запуск_GUI.bat`
3. Откроется окно приложения

## Архитектура

Проект построен по принципам объектно-ориентированного программирования (ООП):

```
┌──────────────────────────────────────────┐
│              main.py                     │
│  ┌─────────────────────────────────────┐ │
│  │       Класс DiscountItem           │ │
│  │  • Название товара (title)          │ │
│  │  • Цена товара                      │ │
│  │  • Процент скидки                   │ │
│  │  • Вложения (attachments)           │ │
│  └─────────────────────────────────────┘ │
│  ┌─────────────────────────────────────┐ │
│  │     Класс DiscountCalculator        │ │
│  │  • Расчёт скидок                    │ │
│  │  • Управление товарами              │ │
│  │  • Управление историей              │ │
│  │  • Бизнес-правила                   │ │
│  └─────────────────────────────────────┘ │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│            file_utils.py                 │
│          Класс FileManager               │
│  • Создание файлов                       │
│  • Чтение файлов                         │
│  • Удаление файлов                       │
└──────────────────┬───────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────┐
│              logger.py                   │
│        Настройка логирования             │
│  • Запись в файл                         │
│  • Вывод в консоль                       │
└──────────────────────────────────────────┘
```

## Бизнес-правила

| Правило | Описание |
|---------|----------|
| Title не пустой | Название товара не может быть пустым или содержать только пробелы |
| Цена > 0 | Цена товара не может быть отрицательной или нулевой |
| Скидка 0-100% | Процент скидки должен быть от 0 до 100 |
| Attachments — список | Вложения должны быть списком строк |
| Нельзя удалять из пустого | Нельзя удалять товары из пустого списка |
| Нельзя редактировать пустое | Нельзя редактировать товары в пустом списке |
| Автосохранение | История сохраняется автоматически при выходе |
| Логирование | Все действия записываются в файл журнала |
